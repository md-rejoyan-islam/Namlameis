Redesign `/company/leadership` into a premium, magazine-style "feature" page using the founder portrait photos from `public/founder/`.

### Layout
**1. Hero** — keep existing `PageHero` ("In their own words.") on its dark navy field. No change.

**2. Leadership feature cards** — replace the current stacked quote cards with an **editorial split layout**, alternating left/right:

```
┌─────────────────────────────────────────────┐
│  ┌──────────┐                                │
│  │          │   A message from the           │
│  │  PORTRAIT │   Founder & CEO      ← amber  │
│  │   PHOTO   │   eyebrow label               │
│  │  (tall)   │                                │
│  │          │   “I founded Namlameis         │
│  │          │    because…”   ← large quote   │
│  └──────────┘                                │
```

- **Two-column grid** (`lg:grid-cols-2`), alternating photo side via an `order` flip on the second card.
- **Portrait photo**: tall aspect (`aspect-[4/5]`), `object-cover`, rounded-2xl (`--radius-2xl` = 32px), ambient `--shadow-card`, subtle hover lift.
- **Quote side**: amber mono eyebrow label ("A message from the Founder & CEO"), then the quote in Season Mix (via `font-display`) at ~`text-2xl/3xl`, normal weight per the site-wide title treatment, black (`text-ink-deep`). Name + role below in a clean caption block with a thin divider.
- Generous vertical rhythm; each card in its own `Reveal` with staggered delay.
- On mobile: single column, photo first then quote.
